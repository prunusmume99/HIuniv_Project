# 📓 results/maps 폴더

## 📋 폴더 개요

이 폴더에는 **사회취약지수**를 기반으로 생성한 Folium 인터랙티브 지도를 HTML로 내보낸 결과물이 들어 있습니다.  
지도에는 행정동별 취약지수가 색으로 표시되며, 우측 상단에는 Chart.js 기반의 Top-10 패널이 붙어 있어 지역별 최대 취약동을 실시간으로 비교할 수 있습니다.
