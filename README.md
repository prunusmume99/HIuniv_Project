# HIuniv_Project
홍익대학교 세종캠퍼스 빅데이터 분석 역량 강화 프로젝트 레포지토리입니다.


# 📘 우선대응지도: 극한 강우와 종합 대응 지수 기반 지역 분석 프로젝트

# 📌 프로젝트 개요

## 📍 프로젝트명  
**우선대응지도: 극한 강우와 종합 대응 지수 기반 지역 분석**

---

## 🧭 목적 및 배경  
최근 몇 년간 기후 위기의 영향으로 **국지성 집중호우와 극한 강우가 빈번하게 발생**하고 있으며,  
이로 인한 **침수, 붕괴, 인명 피해 등 재난 규모 또한 심각하게 확대**되고 있습니다.  
특히 **배수 인프라가 부족하고, 사회적 약자가 밀집한 지역일수록 피해 규모가 더욱 커지는 문제**가 확인되고 있습니다.

본 프로젝트는 이러한 문제의식에서 출발하여,  
- **강수량 데이터 및 기후 지표를 분석하여 극한 강우 위험 지역을 파악하고**,  
- **사회취약 지표를 함께 결합**함으로써  
극한 기상에 **선제적으로 대응이 필요한 지역을 우선순위화**하고,  
이를 통해 **재난 대응 및 행정 정책 수립에 기여할 수 있는 데이터 기반 지도와 인사이트**를 도출하는 것이 목적입니다.

---

## 🔍 분석 내용

### 1. 극한 강우 분석
- 2015~2025년 전국 강수량 데이터를 수집하여 연도별/지역별 극한 강우 발생 패턴 분석  
- 이상치 탐지, 급증 지역 추이 시각화 및 랭킹화

### 2. 사회취약도 분석 (SoVI)
- 고령인구 비율, 기초생활수급자, 1인가구, 장애인 비율 등 인구 통계 기반 사회적 취약성 점수화
- **→ 기초 인프라 부족 또는 재난 취약 계층 집중 지역 파악**

### 3. 하수도 인프라 지수 분석
- 공공하수처리구역 인구보급률, 하수도 설치율, 고도처리인구 비율 등을 종합하여 인프라 구축 수준 및 침수 대응 능력 분석
- 인구 밀도 보정 정규화를 포함한 종합 인프라 점수 산출

### 4. 주거 취약 지수 분석
- 노후주택비율과 자연재해위험도 데이터를 기반으로 주거환경의 물리적 취약성 평가
- 주거 환경의 재해 민감도 및 회복력 저하 지역 도출

### 5. 우선대응 위험도 매트릭스 구축
- 극한강우 점수, 사회취약지수, 주거취약지수, 하수도 인프라 지수를 통합
- 시군구 단위로 위험 등급화하여 지도 시각화
- 결과를 지도 시각화 및 순위 분석 형태로 제공

### 6. 🆕 통합 취약지수 지도 시스템 (최신)
- **4개 레이어 시스템**: 주거취약지수, 수도인프라지수, 사회취약지수, 통합취약지수
- **유연한 매핑 시스템**: 부분 단어 매칭, 시도명 정규화, 키워드 기반 매칭
- **등급별 색상 시스템**: 각 지수별 등급에 따른 직관적인 색상 구분
- **인터랙티브 기능**: 툴팁, 확대/축소, 레이어 컨트롤, 전체화면
- **높은 매핑 성공률**: 94.3% 매핑 성공 (3295개 성공, 200개 실패)
- **특별 기능**: 세종특별자치시 매핑, 부분 매칭, 시도명 정규화

### 7. 🆕 강화된 사용자 경험 (UX) 개선
- **레이어 컨트롤**: 4개 지수를 하나의 인터페이스에서 레이어로 전환
- **상세 툴팁**: 각 지역의 모든 지수 정보를 한 번에 확인
- **전체화면 모드**: 지도 전체화면으로 상세 분석 가능
- **반응형 디자인**: 다양한 화면 크기에 최적화된 레이아웃

## 🆕 최신 주요 성과

### 🗺️ **통합 취약성 지도 시스템**
- **탭 형태 인터페이스**: 사회취약지수, 수도인프라지수, 주거취약지수를 한 페이지에서 확인
- **등급별 색상 시스템**: 각 지수별 등급에 따른 직관적인 색상 구분
- **최강화된 매핑**: 99-100% 매칭률로 정확한 지리적 데이터 연동
- **인터랙티브 기능**: 툴팁, 확대/축소, 레이어 컨트롤 등

### 📊 **등급 분류 시스템**
- **하수도 인프라**: 80/60/40 기준으로 4등급 분류
- **주거취약지수**: 70/50/30/10 기준으로 5등급 분류
- **사회취약지수**: 25/50/75 기준으로 4등급 분류

### 🎯 **데이터 품질**
- **매칭 정확도**: 99-100% 매칭률 달성
- **데이터 완전성**: 모든 시도(17개) 포함
- **시각화 품질**: 인터랙티브 지도 정상 작동

### 🆕 **환경 호환성 및 UX 개선**
- **크로스 플랫폼 지원**: Windows → macOS 경로 변환 완료
- **Git LFS 통합**: 대용량 HTML 파일 정상 처리
- **의존성 관리**: 필요한 Python 패키지 자동 설치
- **마우스 오버 툴팁**: 클릭 없이 지역 정보 확인 가능
- **통합 인터페이스**: 여러 지도를 하나의 시스템에서 관리
